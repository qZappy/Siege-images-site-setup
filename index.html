How to use

  1. Create a public repo. Folder structure: /images/<MapName>/your_image_files.jpg.
  2. Enable GitHub Pages: Settings → Pages → Deploy from branch (main) → root.
  3. Upload this app: index.html, assets/, icons/, manifest.json, sw.js, 404.html.
  4. Open the Pages site on your phone. If auto-detect fails, fill owner/repo inputs or use ?owner=&repo=&branch=.

Loads images via public GitHub API (60 req/hr per IP unauthenticated). Types: jpg, jpeg, png, webp, gif.
